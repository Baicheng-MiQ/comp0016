EyeGaze Requirements

These requiremtns are set on conditions that the users head is in a fixed position 1. Design a calibration page (back end) which allows the user to pick their calibration requirements. Requirements can include, number of points they can calibrate with (1,5,9).

  1. Wink and other feature options should work fluently, the user shpuld have the option of picking how they want to control mouse clicks. Additonally the jitter of the coe given to us should be minimal allowing the mouse to move seamlessly,noise control.

  2. Desing the front end calibration, which allows the user to see themselves and a avatar.The avatar should mimic the face movement and specifically highlight the eye placement.

  3. Create a heat map. This should be based on the nose and eyes, creating a triangle shape between them and measuring the distances in order to find out how far/close the user is from the screen
